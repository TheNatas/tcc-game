# Detetive Sonora

Quick guide to download, install, and run the game.

## Download

Open one of the game pages (Game Jolt or itch.io) and click the "Download" button.

### Game Jolt

![Game Jolt game page with the Download button highlighted](./downloading_instructions/gamejolt1.png)

### itch.io

1) Game page — click "Download Now":

![itch.io game page showing the Download Now button](./downloading_instructions/itchio1.png)

2) Confirmation dialog — click "No thanks, just take me to the downloads":

![itch.io confirmation dialog with the option to proceed to downloads](./downloading_instructions/itchio2.png)

3) Downloads page — click the "Download" button to get the .zip file:

![itch.io downloads page with the zip file Download button](./downloading_instructions/itchio3.png)

## Installation

- Unzip the .zip file (detetive_sonora.zip).

## Run

- Run the .exe file (detetive sonora.exe).

## Notes

- To run, the executable (.exe) must be in the same folder as the .pck file and both must share the same name (detetive sonora.exe and detetive sonora.pck).
